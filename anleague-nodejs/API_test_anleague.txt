=**Signup (POST /auth/signup):
curl -X POST https://anleague-api.onrender.com/auth/signup \
  -H "Content-Type: application/json" \
  -d '{"username": "rep_nigeria", "password": "password123", "email": "rep_nigeria@example.com", "country": "Nigeria", "role": "representative"}'

=**Login (POST /auth/login):
curl -X POST https://anleague-api.onrender.com/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "rep_nigeria", "password": "password123"}'

=**Autofill Team (POST /teams/autofill):
curl -X POST https://anleague-api.onrender.com/teams/autofill \
  -H "Authorization: Bearer <your_jwt_token>" \
  -H "Content-Type: application/json" \
  -d '{"country": "Nigeria"}'

=**Create 8th Team (POST /teams/create):
curl -X POST https://anleague-api.onrender.com/teams/create \
  -H "Authorization: Bearer <your_jwt_token>" \
  -H "Content-Type: application/json" \
  -d '{"country": "South Africa", "manager": "Hugo Broos", "captain_name": "Ronwen Williams", "squad": [/* 23 players */]}'

=**Start Tournament (POST /admin/start):
Create an admin user in Atlas (anleague.users):
{
  "username": "admin",
  "password": "$2a$10$your_hashed_password", // Use bcrypt to hash "admin123"
  "role": "admin",
  "email": "admin@example.com"
}
Login as admin to get JWT.
curl -X POST https://anleague-api.onrender.com/admin/start \
  -H "Authorization: Bearer <admin_jwt_token>" \
  -H "Content-Type: application/json"

=**Simulate Match (POST /admin/simulate):
curl -X POST https://anleague-api.onrender.com/admin/simulate \
  -H "Authorization: Bearer <admin_jwt_token>" \
  -H "Content-Type: application/json" \
  -d '{"match_id": "<match_id_from_tournament>"}'

=**Play Match with AI (POST /admin/play):
curl -X POST https://anleague-api.onrender.com/admin/play \
  -H "Authorization: Bearer <admin_jwt_token>" \
  -H "Content-Type: application/json" \
  -d '{"match_id": "<match_id_from_tournament>"}'

=**Public Views:
GET /bracket: curl https://anleague-api.onrender.com/bracket
GET /match/:id: Use a match_id from anleague.matches.
GET /rankings: Check goal scorers or team rankings.
